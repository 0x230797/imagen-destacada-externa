Agrega una imagen destacada usando una imagen externa.

== Description ==

Este pequeño plugin te ayuda a usar una imagen externa como imagen destacada en tu página web usando sólo la url de la imagen.

== Frequently Asked Questions ==
= ¿El plugin tendrá una versión de paga? = 

No, el plugin es totalmente gratis.
